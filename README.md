Vagrant
=======

My Vagrant template for a virtual machine with latest Docker installation.
 
    sudo apt-get install -y virtualbox vagrant
    git clone 
    vagrant up

Current tested setup:
OSX:

    Vagrant 1.3.5
    Virtualbox 4.3.12  

Debian:

    Kernel 3.14
    Vagrant 1.2.7
    Virtalbox 4.2.24
